# 手势控制3D圣诞树

一个基于Three.js和MediaPipe Hands的手势控制3D圣诞树应用。

## 功能

- ✊ 握拳：回到合拢态
- 🖐️ 五指张开：进入散开态
- 🔄 手旋转：在散开态下调整相机角度
- 🤏 抓取：进入照片放大态

## 技术栈

- Three.js
- MediaPipe Hands
- WebGL
- 自定义Shader
- 后期处理效果

## 运行方法

1. 在项目目录下启动本地HTTP服务器：
   ```
   python -m http.server 8080
   ```

2. 在浏览器中访问：
   ```
   http://localhost:8080
   ```

3. 允许浏览器访问摄像头

4. 使用手势控制圣诞树

## 三种状态

- **合拢态**：所有元素收拢为圆锥体
- **散开态**：元素在空间中无序漂浮
- **照片放大态**：单张照片放大，背景保持散开

## 照片上传

点击左上角的"上传照片"按钮，选择多张照片上传到3D照片云中。
